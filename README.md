# PetrikShop

This is a Spring Boot code example to demonstrate custom GitHub Copilot prompt on comprehensive testing (unit tests, functional tests, integration tests).

You can look at the custom prompt at `.github/prompts` directory.

If you want to use the prompt, add `\comprehensive-tests` before your own description when chatting with GitHub Copilot.
Make sure that you are using Agent mode, so GitHub Copilot can modify your tests.
